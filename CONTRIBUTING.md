# Contributing to DayBrief

Thank you for your interest in contributing to DayBrief!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/daybrief.git`
3. Create a branch: `git checkout -b my-feature`
4. Make your changes
5. Run tests: `make test`
6. Run linter: `make lint`
7. Commit and push your changes
8. Open a Pull Request

## Development

### Prerequisites

- Go 1.25+
- golangci-lint

### Building

```bash
make build
```

### Testing

```bash
make test
```

### Linting

```bash
make lint
```

## Guidelines

- Write tests for new features
- Follow existing code style
- Keep commits focused and descriptive
- Update documentation if needed

## Reporting Issues

Please use [GitHub Issues](https://github.com/yoanbernabeu/daybrief/issues) to report bugs or request features.
